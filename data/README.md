# Data

Large data files (Seurat objects, count matrices) are not stored in this
repository. This directory documents what needs to be downloaded and where
to place it.

## Probe Manifests (required for Figure 2D)

Place the probe manifests here after downloading:

### Flex v1 probe set — `data/vcc_probe_manifest.csv`

The VCC dataset used **Chromium Human Transcriptome Probe Set v1.0.1**
(GRCh38-2020-A). Download from the 10x Genomics support site:

  https://www.10xgenomics.com/support/software/cell-ranger/downloads

Search for "Chromium Human Transcriptome Probe Set v1.0.1". Save the CSV to
`data/vcc_probe_manifest.csv`.

Expected columns: `gene_id, probe_seq, probe_id, included, region`
Expected rows: ~54,580 probes

### Flex v2 probe set — `data/a375_v2_probe_manifest.csv`

The A375 FLEXv2 dataset used **Chromium Human Transcriptome Probe Set v2.0.0**
(GRCh38-2024-A). Download from the 10x Genomics support site:

  https://www.10xgenomics.com/support/software/cell-ranger/downloads

Save the CSV to `data/a375_v2_probe_manifest.csv`.

Expected columns: `gene_id, probe_seq, probe_id, included, region, gene_name`

## Seurat Objects (required for compute scripts)

### VCC dataset — for Figure 1

The VCC (Virtual Cell Challenge) dataset is from the Arc Institute. The merged
Seurat object must be downloaded separately and its path set in
`scripts/figure1/01_setup_vcc_seurat.R`:

  VCC_QS_PATH <- "/path/to/merged_seurat.qs"

### PBMC dataset — for Figure 1I

The 10x Genomics GEM-X 16-plex PBMC dataset (320k cells) is publicly
available from the 10x Genomics website. Set the path to the cellranger
`filtered_feature_bc_matrix.h5` output in
`scripts/figure1/compute/06_setup_pbmc_seurat.R`:

  PBMC_H5_PATH <- "/path/to/filtered_feature_bc_matrix.h5"

### A375 FLEXv2 dataset — for Figure 2

The A375 FLEXv2 ligand dictionary dataset is an in-house dataset from the
Chen lab (Broad Institute) and is **not publicly available**. Researchers
interested in accessing this data should contact the corresponding author.
Set the path in `scripts/figure2/08_setup_a375_seurat.R`:

  A375_QS_PATH <- "/path/to/seurat_A375_flexv2.qs"

### Probe-level Seurat objects — for Figures 2E and 2F

The probe-level Seurat objects for VCC and A375 FLEXv2 (where individual
probes, not genes, are the features) were generated by the Chen lab and are
stored on Chen lab servers. Set paths in the compute scripts:

  scripts/figure2/compute/11_compute_vcc_probe_expression.R
  scripts/figure2/compute/12_compute_a375_probe_expression.R
